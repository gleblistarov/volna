# ВОЛНА — общий тренинг на двоих (сервер)

Функциональные тренировки с кросс-фитом, геймификация и общий прогресс
между двумя телефонами. Backend: Node + Express. Хранилище: JSON-файл на
постоянном Volume. Доступ закрыт кодом комнаты.

## Что внутри
- `server.js` — API + раздача фронтенда
- `public/index.html` — само приложение (один файл)
- `package.json` — запуск `npm start`

## API
- `GET  /api/state`  — вернуть `{names, events}`
- `POST /api/events` — добавить события тренировок (дедуп по `id`)
- `POST /api/names`  — задать имена атлетов
- `POST /api/reset`  — обнулить события
Все запросы требуют заголовок `X-Room-Code`, если задана переменная `ROOM_CODE`.

## Переменные окружения
| Переменная   | Значение                | Зачем                                  |
|--------------|-------------------------|----------------------------------------|
| `ROOM_CODE`  | например `butovo2026`   | код входа в комнату (общий на двоих)   |
| `DATA_DIR`   | `/data`                 | папка постоянного хранилища (Volume)   |
| `PORT`       | (Railway задаёт сам)    | трогать не нужно                       |

## Деплой на Railway

### Вариант A — через GitHub (рекомендую)
1. Создай репозиторий на GitHub, залей туда содержимое этой папки
   (`node_modules` и `data` не нужны — они в `.gitignore`).
2. railway.com → **New Project → Deploy from GitHub repo** → выбери репозиторий.
   Railway сам поймёт, что это Node, поставит зависимости и запустит `npm start`.
3. Сервис → **Variables** → добавь:
   - `ROOM_CODE` = свой код (например `butovo2026`)
   - `DATA_DIR` = `/data`
4. Сервис → **Volumes** → **New Volume**, точка монтирования `/data`
   (это чтобы прогресс не стирался при передеплое).
5. Сервис → **Settings → Networking → Generate Domain** → получишь
   `https://что-то.up.railway.app`.

### Вариант B — через Railway CLI (без GitHub)
```bash
npm i -g @railway/cli
railway login
cd volna-server
railway init        # создать проект
railway up          # задеплоить текущую папку
railway domain      # сгенерировать публичный адрес
```
Затем в дашборде добавь `ROOM_CODE`, `DATA_DIR=/data` и Volume на `/data`
(как в шагах 3–4 варианта A) и сделай **Redeploy**.

## Установка на айфон
1. Открой выданный Railway адрес в **Safari** на каждом телефоне.
2. Введи код комнаты (одинаковый на обоих) — при первом входе одного из вас
   задай имена атлетов, второй телефон подтянет их сам.
3. **Поделиться → На экран «Домой»** → появится иконка, запуск на весь экран.

Прогресс общий: завершил WOD на своём телефоне — у партнёра он появится
в течение ~6 секунд (история, XP, дуэль недели, стрики, награды).
Если на тренировке пропал интернет — результат запомнится локально
и улетит на сервер, как только связь вернётся.
